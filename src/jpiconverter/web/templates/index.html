Notes
  • One CSV is produced per uploaded .JPI, with all flights concatenated.
  • GPS (latitude, longitude, altitude, groundspeed) is decoded when the EDM had a GPS feed at flight time. Samples without a GPS lock leave those cells blank.
  • The LAT/LNG cell format follows the FlySto template (Ndd.mm.ss / Wddd.mm.ss); whether FlySto accepts this exact format is unverified — please report back if uploads fail.
  • Files up to 50 MB are accepted.
